# Flashover Version 0.1.1 - Pre-Report-Fix

This version includes the initial implementation of the report logging and viewing feature. However, the feature is not yet fully functional and contains known bugs.

## Key Features & Changes:

- **Report Viewing UI:** Added a "View Past Reports" screen and a modal to display report details.
- **Report API Routes:** Implemented backend routes to save and retrieve report data.

## Known Issues:

- **Report Logging:** The functionality for saving and viewing reports is not working as expected and needs to be debugged and fixed.
- **Event Listeners:** There were several issues with JavaScript event listeners that were addressed, but the report functionality remains broken.
